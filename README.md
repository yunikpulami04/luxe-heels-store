MAISON LUXE — Premium Heels Boutique UI

A sleek and modern eCommerce front-end built with HTML, CSS, and JavaScript. 
Designed with a minimal luxury aesthetic, this project features a responsive grid layout, dynamic cart functionality, and a smooth checkout experience using PayPal integration.

Features:
- Responsive product grid (3 desktop / 1 mobile)
- Clean and elegant UI design
- LocalStorage-based cart system
- PayPal checkout integration
- Fully mobile-friendly layout
